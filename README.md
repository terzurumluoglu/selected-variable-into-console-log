## Insert console.log()

Keyboard Shortcut:

- Mac: `Cmd + Shift + L`
- Windows: `Ctrl + Alt + L`

```js
console.log("variable", variable); // selection
console.log("this.data", this.data); // copied
console.log("HERE"); // no selection and no copied
```
